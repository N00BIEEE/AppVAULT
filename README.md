# 🔐 AppVault – Android Privacy & Security Manager

AppVault is an Android security application designed to help users understand and manage the privacy risks of installed applications. It analyzes permissions, categorizes application risk levels, scans APK files before installation, and monitors recent app activity to improve device security.

---

## 🚀 Features

- 📱 Analyze installed application permissions
- 🔍 Scan APK files before installation
- 🛡️ Risk categorization (Safe, Moderate, Dangerous)
- 📊 Permission-based security analysis
- 📋 Recent app activity monitoring
- ⚡ Fast and intuitive user interface
